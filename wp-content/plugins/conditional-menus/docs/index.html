Conditional Menus

Conditional Menus is a simple yet useful WordPress plugin developed by Themify, which allows you to swap the menus in the theme as per specific conditions. In short, you can have different menus in different posts, pages, categories, archive pages, etc. It works with any WordPress theme that uses the standard WordPress menu function.

How To Use It

The plugin is very simple to use. Once you activate the plugin, you will see the conditional menus on the Manage Locations tab located in your WP Admin > Appearance > Menus page.

  1. To add conditional menu: click "Conditional Menu" and select a menu from the list (you can create these menus in the "Edit Menus" tab)
    • You can remove the menu by selecting "Disable Menu" from the list.
  2. Click on "Conditions" to add conditions in the modal box (tick the checkboxes where you want the menu to appear)
  3. To remove the conditional menus, click on the "X" button
